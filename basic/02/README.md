# 맥 용어
맥에서 사용되는 용어
---
|Windows|Mac|설명|
|:-:|:-:|-|
|Alt 키|Option 키|특수 문자 : 문자키 or Option 키와 문자키 or Option 키와 Shif 키와 문자키를 길게 입력<br>단, 키 반복이 끔으로 되어있으면 작동X<br>ex) é를 입력하려면 Option-E를 누른 다음 E 키를 다시 입력|
|Alt 키|Control-F2|화면 상단의 메뉴 막대에서 키보드를 사용하여 메뉴 오픈 : fn-Control-F2-화살표로 메뉴 선택<br>선택된 메뉴 오픈 : Return<br>메뉴 옵션 선택 : 화살표와 Return 키 사용|
|Alt-Tab|Command-Tab|열려 있는 앱간의 빠른 전환 : Command-Tab<br>원하는 앱이 선택되면 키에서 손 떼기|
|Alt-Tab|Exposé|현재 앱에서 열려 있는 윈도우 모두 표시 : Dock에서 해당 앱의 아이콘을 Control-클릭-모든 윈도우 보기<br>다른 앱 윈도우를 탐색 : Tab|
|닫기 버튼|닫기 버튼|윈도우 닫기 : 윈도우의 왼쪽 상단 모서리에서 닫기 버튼 클릭|
|Control 키|Command 키|동작 또는 단축키를 수행 : 키보드 조합으로 Command 키 사용<br>ex) Command-S : 문서나 파일 저장|
|제어판|시스템 설정|Mac 사용자화 : Apple 메뉴 > 시스템 설정<br>[시스템 설정으로 Mac 사용자화하기](https://support.apple.com/ko-kr/guide/mac-help/mh15217/15.0/mac/15.0)<br>ex) Dock의 크기 및 위치 변경, 라이트/다크 모드, 다른 데스크탑 사진 사용 등|
|Cortana|Siri|Siri를 사용하여 앱을 열거나, Mac 또는 인터넷에 있는 항목을 찾는 등의 작업 수행<br>[Siri 사용하기](https://support.apple.com/ko-kr/guide/mac-help/mchl6b029310/15.0/mac/15.0)|
|장치 관리자|시스템 정보|시스템 정보 앱 : Mac 하드웨어 및 소프트웨어에 대한 상세한 정보 제공<br>Option 키를 누른 상태로 Apple 메뉴 > 시스템 정보 선택|
|디스크 드라이브 추출하기 버튼|미디어 추출 키|Finder에서 해당 디스크를 선택한 다음 파일 > 추출 선택|
|끝내기|종료|앱에서 종료 : 앱 메뉴에서 종료 선택(앱 메뉴는 해당 앱의 이름으로 표기)|
|3D 뒤집기|Mission Control|[Mission Control에서 열려 있는 모든 윈도우를 표시](https://support.apple.com/ko-kr/guide/mac-help/mh35798/15.0/mac/15.0)하려면 Control-위쪽 화살표|
|가젯|위젯|위젯을 사용하여 주식을 살펴보거나 날씨를 확인하는 등의 작업 수행 가능<br>[위젯 추가 및 사용자 정의하기](https://support.apple.com/ko-kr/guide/mac-help/mchl52be5da5/15.0/mac/15.0)|
|Microsoft Photo Editor|사진|사진 앱을 사용하여 iOS 또는 iPadOS 기기나 카메라에서 [사진 및 비디오 가져오기](https://support.apple.com/ko-kr/guide/photos/phta58cd90d3/mac), 편집, 공유하는 등 작업 가능|
|내 컴퓨터, 이 PC|Finder에서 이동 > 컴퓨터를 선택하십시오.|Mac에 연결된 디스크, 광학 디스크 드라이브에 넣은 CD와 DVD, 연결된 네트워크 볼륨 및 사용자에게 있는 모든 디스크 파티션 표시|
|내 문서, 문서 폴더|문서 폴더|문서 저장 : 문서 폴더 사용<br>문서 폴더를 표시하려면 Finder 윈도우를 연 다음, 사이드바에서 ‘문서’ 클릭|
|내 사진, 사진 폴더|그림 폴더|그림 폴더는 홈 폴더 안에 위치. Finder에서 이동 > 홈 선택|
|내 최근 문서|최근 사용 항목|앱과 파일을 열면, 해당 앱 및 파일의 이름이 Apple 메뉴의 최근 사용 항목 목록에 유지<br>파일 메뉴에 '최근 사용 열기' 명령이 있어서 최근에 작업한 문서 목록 조회 가능|
|네트워크 연결|네트워크 설정|네트워크 설정 구성 : Apple 메뉴 > 시스템 설정 > 사이드바에서 네트워크를 클릭(아래로 스크롤)<br>[무선 진단](https://support.apple.com/ko-kr/guide/mac-help/mchlf4de377f/15.0/mac/15.0)을 사용하여 이미 설정한 네트워크의 문제 해결 가능|
|알림 영역, 시스템 트레이|상태 메뉴|상태 메뉴 : 메뉴 막대의 오른쪽에 아이콘 위치, 무선 네트워크 연결 및 Mac 랩탑의 배터리 상태 등을 확인<br>[제어 센터 설정](https://support.apple.com/ko-kr/guide/mac-help/mchlad96d366/15.0/mac/15.0)을 사용하여 메뉴 막대에 표시되는 상태 메뉴 변경|
|화면상의 키보드(OSK)|키보드 뷰어|키보드 뷰어 : 화면 상단의 메뉴 막대 > 입력 메뉴 > 키보드 뷰어 보기 선택<br>입력 메뉴 : 이모티콘 및 기호 보기 아이콘 또는 현재 활성 입력 소스를 나타내는 아이콘으로 식별<br>입력 메뉴가 보이지 않는 경우, Apple 메뉴 > 시스템 설정 > 사이드바에서 키보드 클릭 > 입력 소스 옆의 편집 클릭 > ‘메뉴 막대에서 입력 메뉴 보기’|
|성능 모니터|활성 상태 보기|Mac이 어떻게 기능을 수행하고 어떤 프로세스를 실행하고 있는지 알아보려면, 활성 상태 보기 앱(응용 프로그램 폴더의 유틸리티 폴더에 위치)|
|화면 프린트|Shift-Command-3<br>Shift-Command-4<br>Shift-Command-5|전체 화면 캡처 : Shift-Command-3<br>화면 일부 캡처 : Shift-Command-4-포인터 드래그하여 영역을 선택<br>스크린샷 앱(옵션 설정) : Shift-Command-5|
|프린터 및 스캐너|프린터 및 스캐너 설정|프린터 선택 및 설정 : Apple 메뉴 > 시스템 설정 > 사이드바에서 프린터 및 스캐너 클릭|
|속성|정보 가져오기|파일, 폴더, 디스크, 서버 또는 기타 항목에 관한 정보 표시 : Finder에서 항목 선택 > 파일 > 정보 가져오기<br>정보 윈도우에서 항목에 대한 소유권 및 권한 설정<br>파일을 열고자 하는 앱도 선택 가능|
|휴지통|휴지통|파일 및 폴더 삭제 : Dock에 있는 휴지통으로 드래그<br>항목 완전히 제거 : 데스크탑 클릭 > Finder > 휴지통 비우기|
|검색|Spotlight|파일, 이메일 및 기타 항목 검색([Spotlight로 검색하기](https://support.apple.com/ko-kr/guide/mac-help/mchlp1008/15.0/mac/15.0))<br>Finder, Mail 및 연락처 등 여러 앱은 도구 막대에 검색 필드가 있어 해당 앱에 있는 항목을 빠르게 찾기 가능|
|단축어|가상본|가상본(바로가기) 생성 : 파일 또는 앱 선택 > 파일 > 가상본 만들기|
|스냅 윈도우|윈도우 타일 정렬|윈도우를 화면 가장자리로 드래그하거나 윈도우의 왼쪽 상단 모서리에 있는 초록색 버튼을 사용하여 레이아웃 선택<br>[윈도우 정렬하기](https://support.apple.com/ko-kr/guide/mac-help/mchlef287e5d/15.0/mac/15.0)|
|대기 또는 절전|잠자기|잠자기 : 저전력 모드<br>설정 방법 : Apple 메뉴 > 잠자기|
|시작 메뉴|Launchpad|모든 앱 및 유틸리티에 빠르게 접근 : Dock에서 Launchpad 아이콘 클릭
|작업 관리자|활성 상태 보기|Mac이 어떻게 기능을 수행하고 어떤 프로세스를 실행하고 있는지 알아보려면, 활성 상태 보기 앱(응용 프로그램 폴더의 유틸리티 폴더에 위치)|
|작업 표시줄|Dock|[Dock을 사용](https://support.apple.com/ko-kr/guide/mac-help/mh35859/15.0/mac/15.0)하여 자주 사용하는 앱, 파일, 폴더 및 웹사이트 열기 가능<br>기본적으로, Dock은 화면의 아래쪽에 위치<br>파일 또는 폴더를 Dock에 추가하려면 아이콘을 Dock의 분리자 오른쪽으로 드래그|
|Windows 탐색기|Finder|파일, 폴더 및 앱 구성 : Finder 사용<br>Finder 윈도우를 열려면 데스크탑 클릭 > 파일 > 새로운 Finder 윈도우|
|Windows Media Player|QuickTime Player<br>음악|동영상 및 음악 재생 : QuickTime Player 앱 사용<br>음악 감상, iTunes Store에서 음악 구입, Apple Music 구독, 개인 디지털 음악 보관함 생성 등|
|Windows MovieMaker|iMovie|iMovie 앱 : 디지털 비디오 카메라로부터 비디오 다운로드, 개인 동영상 생성 등|

Mac 키보드상의 Windows 키
---
|Windows 키|Mac 키|설명|
|:-:|:-:|-|
|Alt|Option|é와 같은 특수 문자 입력|
|제어기<br>Windows 키|Control<br>Command|Control 키를 누른 상태로 클릭하면 단축 메뉴 오픈<br>문서 또는 파일을 저장하려면 Command-S|
|Home 및 End|Command-왼쪽 화살표 또는 오른쪽 화살표<br>Fn–왼쪽 화살표 또는 오른쪽 화살표|Command-왼쪽 화살표 : 삽입 지점을 현재 줄의 시작으로 이동<br>Command-오른쪽 화살표 : 삽입 지점을 현재 줄의 끝으로 이동<br>Fn–왼쪽 화살표(Home) : 현재 문서의 시작으로 스크롤<br>Fn–오른쪽 화살표(End) : 현재 문서의 끝으로 스크롤|
|Num Lock|Num Lock<br>Shift-Clear|(일부 앱)숫자 키패드에 있는 키를 누를 때 숫자를 입력할지, 포인터를 이동할지 조절|
|Scroll Lock|Control-F14<br>(Fn 키를 먼저 길게 눌러야 할 수 있음)|(일부 앱)화살표 키를 누를 때 포인터를 움직일지, 윈도우에서 스크롤할지 조절|
|화면 프린트|Shift-Command-3<br>Shift-Command-4|화면 캡처<br>Shift-Command-3 : 전체 화면 캡처, Shift-Command-4 : 화면의 일부 캡처|
